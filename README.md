# Bank System UI (C++) 🏦

A comprehensive banking management system built with **C++**, demonstrating advanced software engineering principles including **Object-Oriented Programming (OOP)**, security, and data persistence.

## 📌 Project Overview
This project simulates a real-world banking environment through a Command-Line Interface (CLI). It provides a secure platform for managing clients, users, and financial transactions while maintaining data integrity through file-based storage.

## ✨ Key Features
*   **Role-Based Access Control (RBAC):** Implementation of a permission system to restrict access based on user roles.
*   **Security & Privacy:** Passwords are encrypted before storage to ensure user data protection.
*   **Comprehensive Management:** Full CRUD operations (Create, Read, Update, Delete) for both clients and system users.
*   **Banking Operations:** Supports essential financial transactions such as deposits, withdrawals, and balance inquiries.
*   **Data Persistence:** Efficient use of File Handling to save and retrieve data from local files.

## 🛠 Tech Stack & Core Concepts
*   **Language:** C++
*   **Paradigm:** Object-Oriented Programming (Classes, Encapsulation, Modular Design).
*   **Storage:** File Handling (Stream I/O).
*   **Security:** Basic Data Encryption/Decryption algorithms.
